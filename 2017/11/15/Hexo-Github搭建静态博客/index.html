开始吧

首先你需要准备一下：

  1. 一个可以用的github账号，注册地址
  2. 安装git软件,32或64取决于你的电脑，下载地址，好像有的时候这个网站登陆不了，可以直接在百度下载…
  3. Node.js,32或64取决于你的电脑系统，下载地址
  4. 一台差不多的电脑

准备完毕后，好，开搞。
