假装是个标题

首先说一下为什么写这么个博客，在网上你可以搜到一大堆教你如何用hexo搭建博客的教程。

那我为什么还要写呢？

很简单，我是为了练习一下markdown语法、顺便记录一下自己搭博客的心路历程吧，虽然挺简单的。

我想先吐槽一下，为啥我的段落行间距这么大呢，看了一下我用的主题css文件里，p标签的margin属性上下不是0，这主题就这样吧，保持原汁原味吧不改了。

开始吧

首先你需要准备一下：

  1. 一个可以用的github账号，注册地址
  2. 安装git软件，下载地址
  3. 一台差不多的电脑
  4. 一个还算清醒的大脑

准备完毕后，好，开搞。
